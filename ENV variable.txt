** <<<>>> env is an Arry of key value list :

env:                    <<<< from static Values
  - name: VAR_key
    value: VAR_value

env:                    <<<< valueFrom CM
  - name: VAR_key
    valueFrom:
       configMapKeyRef:
           name: CM_name
           key:  key_name_in_CM

env:                   <<<< valueFrom scret
  - name: VAR_key
    valueFrom: 
        secretKeyRef:
           name: secret_name
           key:  key_name_in_secret

*** <<<<<>>>>> Also see importing All CM an secret __vars__  :( inject all data ):

envFrom:
  - configMapRef:
       name: name_of_the_CM
envFrom:
  - secretRef:
       name: name_of_the_secret


** >>>  Volume also can be From CM or secret:

    volumeMounts:
    - name: foo
      mountPath: "/etc/foo"
      readOnly: true
  volumes:
  - name: foo
    secret:
      secretName: mysecret
      optional: false             # default setting; "mysecret" must exist
